# benshattuck.studio

Personal portfolio/promotional site for my PM/PO job search. Plain HTML/CSS/JS, no build step, deployed via GitHub Pages at [benshattuck.studio](https://benshattuck.studio).

## Structure

- `index.html` — home
- `about.html` — career narrative
- `projects.html` — case studies
- `resume.html` — resume (embedded PDF + download)
- `contact.html` — email, LinkedIn, GitHub
- `css/style.css` — styles
- `js/main.js` — scripts (minimal)
- `assets/` — images, resume PDF

## Local development

No build step. Open `index.html` directly in a browser, or serve it:

```
npx serve .
```
